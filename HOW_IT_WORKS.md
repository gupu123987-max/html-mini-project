# วิธีการทำงานของเว็บไซต์ข่าว

## ภาพรวมของโปรเจค

เว็บไซต์ข่าวนี้เป็นเว็บไซต์ที่แสดงข่าวสารและให้ผู้ใช้สามารถเพิ่มข่าวใหม่ๆ ได้ด้วยตัวเอง มันถูกสร้างขึ้นด้วยเทคโนโลยีพื้นฐาน 3 อย่าง:

1. **HTML** - โครงสร้างและเนื้อหาของเว็บไซต์
2. **CSS** - การจัดรูปแบบและดีไซน์ให้สวยงาม
3. **JavaScript** - ฟังก์ชันการทำงานต่างๆ

## ส่วนประกอบหลัก

### 1. หน้าเว็บหลัก (index.html)

นี่คือไฟล์หลักที่เป็นตัวกำหนดโครงสร้างของเว็บไซต์ทั้งหมด มันเหมือนกับ "แบบบ้าน" ที่บอกว่า:
- ส่วนหัวอยู่ตรงไหน
- ส่วนเนื้อหาอยู่ตรงไหน
- ส่วนท้ายอยู่ตรงไหน
- มีปุ่มอะไรบ้าง
- มีช่องกรอกข้อมูลอะไรบ้าง

**ส่วนสำคัญๆ ในหน้านี้:**
- **ส่วนหัว (Header)** - มีชื่อเว็บไซต์และปุ่มสำหรับแอดมิน
- **ส่วนเนื้อหา (Main Content)** - แสดงรายการข่าวทั้งหมด
- **ส่วนฟอร์ม (Form)** - ซ่อนอยู่ ใช้สำหรับเพิ่มข่าวใหม่
- **ส่วนท้าย (Footer)** - ข้อมูลเพิ่มเติม

### 2. ไฟล์สไตล์ (css/style.css)

ไฟล์นี้ทำหน้าที่ "ตกแต่งบ้าน" ให้สวยงาม โดยกำหนด:
- สีของตัวอักษรและพื้นหลัง
- ขนาดของตัวอักษร
- รูปร่างของปุ่ม
- การจัดวางตำแหน่งต่างๆ
- เอฟเฟกต์ต่างๆ เช่น เงา รูปร่าง เป็นต้น

### 3. ไฟล์สคริปต์ (js/app.js และ js/news.js)

ไฟล์เหล่านี้ทำหน้าที่เป็น "สมอง" ของเว็บไซต์ ทำให้เว็บไซต์สามารถทำงานได้

**js/app.js - ควบคุมการมองเห็น**
- ซ่อน/แสดงส่วนต่างๆ ของเว็บไซต์
- จัดการการล็อกอินของแอดมิน
- ควบคุมการเปิด/ปิดฟอร์มเพิ่มข่าว

**js/news.js - จัดการข่าวสาร**
- เพิ่มข่าวใหม่
- ลบข่าว
- แสดงข่าวทั้งหมด

## วิธีการทำงานทีละขั้นตอน

### 1. เมื่อเปิดเว็บไซต์

1. เบราว์เซอร์อ่านไฟล์ `index.html` ก่อน
2. เบราว์เซอร์โหลดไฟล์ `css/style.css` เพื่อจัดรูปแบบ
3. เบราว์เซอร์โหลดไฟล์ `js/app.js` และ `js/news.js` เพื่อให้เว็บไซต์ทำงาน
4. เว็บไซต์จะพยายามโหลดข่าวที่บันทึกไว้จากเบราว์เซอร์ (ถ้ามี)
5. ถ้าไม่มีข่าว ก็จะแสดงข่าวตัวอย่าง

### 2. การเพิ่มข่าวใหม่

1. แอดมินล็อกอินเข้าสู่ระบบ
2. กดปุ่ม "เพิ่มข่าว"
3. ฟอร์มจะปรากฏขึ้น
4. แอดมินกรอกหัวข้อและเนื้อหาข่าว
5. กดปุ่ม "บันทึก"
6. JavaScript จะ:
   - ตรวจสอบว่ากรอกข้อมูลครบหรือยัง
   - สร้างข่าวใหม่และเพิ่มลงในรายการ
   - บันทึกข้อมูลลงในเบราว์เซอร์
   - แสดงข่าวใหม่บนหน้าเว็บ
   - ซ่อนฟอร์มกลับ

### 3. การลบข่าว

1. แอดมินล็อกอินเข้าสู่ระบบ
2. กดปุ่ม "ลบ" ใต้ข่าวที่ต้องการลบ
3. ระบบจะถามยืนยัน
4. ถ้ากดยืนยัน JavaScript จะ:
   - ลบข่าวออกจากรายการ
   - อัพเดทข้อมูลในเบราว์เซอร์
   - รีเฟรชการแสดงผล

### 4. การบันทึกข้อมูล

เว็บไซต์นี้ใช้เทคโนโลยีที่ชื่อว่า **localStorage** ของเบราว์เซอร์ ซึ่งเป็นเหมือน "กล่องเก็บของ" ในเบราว์เซอร์ที่สามารถเก็บข้อมูลไว้ได้แม้จะปิดเว็บไซต์ไปแล้ว

เมื่อเพิ่มข่าว:
1. ข่าวจะถูกแปลงเป็นรูปแบบที่คอมพิวเตอร์เข้าใจได้
2. บันทึกลงใน localStorage ด้วยชื่อ "myNews"
3. เมื่อเปิดเว็บไซต์ใหม่ ก็จะอ่านข้อมูลจาก localStorage แล้วแสดงผล

## เทคโนโลยี Docker

เว็บไซต์นี้ยังสามารถรันผ่าน Docker ได้ ซึ่งเป็นเหมือน "กล่องปิด" ที่ทำให้เว็บไซต์ทำงานได้ในทุกเครื่องโดยไม่ต้องติดตั้งอะไรเพิ่ม

**Docker ประกอบด้วย:**
- **Dockerfile** - คำสั่งในการสร้างกล่อง Docker
- **docker-compose.yml** - คำสั่งในการรันกล่อง Docker
- **.dockerignore** - ไฟล์ที่ไม่ต้องการใส่ในกล่อง

**ข้อดีของ Docker:**
- รันได้ทุกเครื่องเหมือนกัน
- ไม่ต้องติดตั้งโปรแกรมเพิ่ม
- แยกจากโปรแกรมอื่นๆ ในเครื่อง

## การใช้งานพื้นฐาน

### สำหรับผู้ใช้ทั่วไป
1. เปิดเว็บไซต์
2. อ่านข่าวที่แสดง
3. ไม่สามารถเพิ่มหรือลบข่าวได้

### สำหรับแอดมิน
1. เปิดเว็บไซต์
2. กดปุ่ม "แอดมิน"
3. กรอกรหัสผ่าน: `123`
4. กดปุ่ม "เข้าสู่ระบบ"
5. จะเห็นปุ่ม "เพิ่มข่าว" และ "ลบ" ใต้ข่าวต่างๆ
6. สามารถเพิ่มข่าวใหม่หรือลบข่าวเก่าได้

## ข้อจำกัด

เว็บไซต์นี้มีข้อจำกัดบางอย่างเพราะใช้เทคโนโลยีพื้นฐาน:

1. **ข้อมูลเก็บในเบราว์เซอร์** - ถ้าลบข้อมูลเบราว์เซอร์ ข่าวจะหายไป
2. **ไม่มีการจัดการผู้ใช้** - ทุกคนที่รู้รหัสผ่านสามารถเป็นแอดมินได้
3. **ไม่มีการยืนยันความถูกต้อง** - ไม่มีการตรวจสอบว่าข่าวเป็นจริงหรือไม่
4. **ไม่มีการสำรองข้อมูล** - ถ้าข้อมูลเสียหาย กู้คืนไม่ได้

## สรุป

เว็บไซต์ข่าวนี้เป็นตัวอย่างการสร้างเว็บไซต์ด้วย HTML, CSS และ JavaScript พื้นฐาน มันแสดงให้เห็นว่าแม้จะใช้เทคโนโลยีพื้นฐาน ก็สามารถสร้างเว็บไซต์ที่ใช้งานได้จริงและมีฟังก์ชันครบถ้วน

เว็บไซต์นี้เหมาะสำหรับ:
- แสดงข่าวภายในองค์กร
- บล็อกส่วนตัว
- เว็บไซต์ข่าวขนาดเล็ก
- ตัวอย่างการเรียนรู้การพัฒนาเว็บไซต์
